os เป็นโมดูลในภาษา Python ที่มาพร้อมกับการติดตั้งและใช้งานอยู่แล้ว (built-in module) ซึ่งให้ฟังก์ชันและเมทอดต่างๆ เพื่อทำงานกับระบบปฏิบัติการ (Operating System) ของเครื่องคอมพิวเตอร์

โดยโมดูล os มีบางฟังก์ชันที่ใช้งานบ่อย ๆ ดังนี้:

os.system(command): ใช้เพื่อทำงานคำสั่งในระบบปฏิบัติการ โดยการเรียกใช้คำสั่งที่สามารถทำงานในระบบปฏิบัติการผ่านพารามิเตอร์ command และสามารถรับค่าออกมาได้ (อย่างไรก็ตาม ในรุ่น Python 3.5 ขึ้นไป จะแนะนำให้ใช้ subprocess แทน os.system())

os.listdir(path): ใช้เพื่อแสดงรายการไฟล์และโฟลเดอร์ที่อยู่ในตำแหน่งที่ระบุโดยพารามิเตอร์ path

os.getcwd(): คืนค่าสตริงที่บ่งชี้ไปที่ไดเรกทอรีปัจจุบัน (Current Working Directory)

os.mkdir(path): ใช้สร้างโฟลเดอร์ใหม่ที่ตำแหน่งที่ระบุในพารามิเตอร์ path

os.remove(path): ใช้ลบไฟล์ที่ตำแหน่งที่ระบุในพารามิเตอร์ path

os.path: เป็นโมดูลย่อยภายใน os ที่ให้ฟังก์ชันและเมทอดที่เกี่ยวข้องกับการทำงานในระบบไฟล์ เช่น os.path.exists(), os.path.isdir(), os.path.isfile() เป็นต้น

แต่ละฟังก์ชันใน os ใช้งานควบคู่กับระบบปฏิบัติการที่ Python ทำงานอยู่ โมดูล os ทำให้สามารถทำงานกับระบบไฟล์ สร้างโฟลเดอร์ และปรับแต่งตำแหน่งไฟล์ได้ง่ายขึ้นในภาษา Python โดยไม่ต้องเรียกใช้คำสั่งหรือโปรแกรมภายนอก นอกจากนี้ os ยังมีฟังก์ชันอื่นๆ ที่ช่วยในการควบคุมระบบปฏิบัติการและทำการจัดการไฟล์และโฟลเดอร์อย่างหลากหลายตามความต้องการของคุณ
